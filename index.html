Como pegar sua chave de API

É gratuito, leva menos de 2 minutos e não exige cartão de crédito.

Passo a passo
  1. 1 Acesse aistudio.google.com/app/apikey
  2. 2 Faça login com a sua conta Google
  3. 3 Clique em "Create API Key" e selecione ou crie um projeto
  4. 4 Copie a chave gerada — ela começa com AIzaSy...
  5. 5 Cole no campo do Fólio e clique em Salvar
Por que é necessário?

O Fólio usa o modelo de visão do Google Gemini para ler e transcrever o texto das páginas escaneadas. Para fazer isso, ele precisa se identificar à API do Google — e é isso que a chave faz. Sem ela, o Google não sabe quem está fazendo as requisições.

É seguro?
🔒
Fica só no seu dispositivo
A chave é salva apenas no localStorage do seu navegador. Nenhum servidor do Fólio a recebe.
📡

As imagens e a chave saem do seu navegador direto para a API do Google. Não passam por intermediários.
