Sobre o projeto

O que é o Epubly?

Uma ferramenta gratuita e de código aberto para resgatar livros esquecidos.

A iniciativa
Milhares de obras raras existem apenas como scans de baixa qualidade.
Como funciona
O PDF é renderizado direto no navegador e as imagens vão para a IA.
Código aberto
Todo o código está disponível no GitHub.
Por que as doações são fundamentais
Cada conversão tem um custo real de IA. Sem doações o projeto precisa fechar.
